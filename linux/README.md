# FunnyPets для Linux (Debian 13+)

Порт macOS-версии на Python + GTK3. Один файл, без сборки.

## Установка

```sh
sudo apt install python3-gi gir1.2-gtk-3.0 libxss1
python3 funnypets.py
```

## Автозапуск

```sh
mkdir -p ~/.local/share/applications ~/.config/autostart
cp funnypets.desktop ~/.config/autostart/
# поправь Exec= на свой путь к funnypets.py
```

## Что умеет

- Кот или хомяк (у хомяка надуваются щёчки при печати)
- 5 окрасов + 4 узора (сиамский, полосатый, калико, смокинг)
- Глаза следят за курсором, моргает, дёргает хвостом
- Месит лапками при печати, перегревается при долгой печати
- Погладь по голове — мурчит с сердечками
- Засыпает при простое (ночью быстрее)
- Помодоро-таймер и напоминания о разминке
- Интеграция с Claude Code: тот же `~/.comnyan/agent`
  (думает вместе с агентом, прыгает когда агент закончил)
- Имя питомца под спрайтом, позиция запоминается

## Ограничения

- Нужен **X11** (или сессия «GNOME on Xorg»). На чистом Wayland окно
  нельзя позиционировать, а глобальный простой ввода недоступен —
  часть реакций отключится.
- Детекция печати эвристическая (через X ScreenSaver idle): активность
  без движения мыши считается печатью.
- Нет: охоты за курсором, рулона бумаги, peek-режима, импорта окрасов,
  звуков. Спрайты общие с macOS-версией (`sprites.json` генерируется
  из `../sprites.py`).
